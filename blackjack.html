Blackjack

A fully playable card game with betting logic and UI flow

Projects Experience Certifications Bio Contact

Blackjack is a 2D card game where players compete against a dealer using classic casino rules. The game supports betting, hitting, standing, bust conditions, and dealer AI. It features simple but effective UI, sound feedback, and a responsive game loop that lets players restart and try different outcomes.

Key Features

  • Standard Blackjack rules and win/loss detection
  • Chip-based betting and dynamic UI updates
  • Dealer AI that draws according to house rules
  • Restart and game over handling with animations and sound

Development Skills Demonstrated

  • Game Logic: Built full card/deck system with win conditions and betting math
  • Unity UI: Managed player input, score display, and feedback with animations
  • C# Scripting: Turn handling, dealer behavior, and deck shuffling
