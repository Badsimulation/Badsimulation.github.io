Space Shooter

An arcade-style vertical shooter built in Unity

Projects Experience Certifications Bio Contact

Space Shooter is a fast-paced vertical-scrolling arcade game built in Unity. Players control a spaceship dodging enemy fire, collecting power-ups, and blasting through waves of enemies. Designed as a complete arcade loop with win/lose states, the project focuses on responsive controls and game flow polish.

Key Features

  • Wave-based enemy spawning with increasing difficulty
  • Extra life pickups, score tracking, and level progression
  • Scrolling background and animated transitions between stages
  • Boss encounter and final victory condition

Development Skills Demonstrated

  • Unity 2D Game Framework: Game loop with win/lose logic, enemy spawners, and transitions
  • C# Programming: Player input handling, health/damage systems, power-ups, and boss logic
  • UI Implementation: Lives and score display, level intro text, and leaderboard integration
  • Scene polish: Used coroutines and easing functions for scene transitions and effects
