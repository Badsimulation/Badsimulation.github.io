Flappy Cat

A simple twist on a classic tap-to-fly arcade game

Projects Experience Certifications Bio Contact

Flappy Cat is a lighthearted arcade-style game inspired by the original Flappy Bird. You control a pixel cat trying to flap through endless pipe obstacles, gaining points with each pass. A relaxing color palette and cat-themed visuals give it a fun, casual feel while retaining the challenge of timing-based gameplay.

Key Features

  • Tap-to-fly mechanic using Unity's physics engine
  • Endless scrolling pipe obstacle generation
  • Score tracking and game over UI
  • Clean reset and retry flow

Development Skills Demonstrated

  • Unity 2D Physics: Rigidbody jump force, gravity, and collision
  • C# Scripting: Procedural obstacle spawning, scoring, and scene management
  • UI Design: Minimalist score display, restart prompt, and screen transitions
