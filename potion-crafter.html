Oops! All Alchemy

A 3D alchemy game focused on experimentation and discovery

Projects Experience Certifications Bio Contact

Oops! All Alchemy is a 3D potion-brewing simulator where players combine ingredients in a cauldron to create magical effects. With a focus on experimentation, players discover new recipes through trial and error, supported by dynamic visual feedback like colored liquids, sparkles, and foam.

Key Features

  • Drag-and-drop 3D ingredient system
  • Heat-based potency mechanic with ingredient destruction
  • Journal system that tracks discovered potions
  • Fully dynamic visual liquid feedback

Development Skills Demonstrated

  • 3D Unity Development: Interactive object manipulation and cauldron systems
  • C# Scripting: Ingredient logic, combination effects, and journal data
  • Visual Feedback: Custom shaders, particle systems, and foam/sparkle effects (WIP)
